## Projects decision

**Option A (chosen):** Projects are admin-managed groupings. Sprints belong to a project; tasks inherit project via sprint. UI will add `ProjectManager` + selector (PR 3).

**Option B (rejected):** Hide projects from product narrative — conflicts with maintainer feedback (“if we use projects, UI must reflect it”).
