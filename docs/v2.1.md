## Goals

1. **Design is a deliverable** — restrained vaporwave: deep ink surfaces, magenta accent, modern sans + mono metadata, obvious glow-up within 10 seconds.
2. **Real collaboration** — workspace team roster in **Postgres** (not `localStorage`); stable assignee identity across users and browsers.
3. **Security unchanged in spirit** — **RLS enforces**; UI gating is UX only. Use RPC / security definer where needed.

## Visual principles (strict)

- **Ink surfaces, clear hierarchy** — depth via `--bg-0` / `--bg-1` / `--bg-2`, not random gradients.
- **Magenta is accent, not body text** — default copy uses `--text-0` / `--text-1`; `--accent-0` for focus, primary actions, chips.
- **One glow rule** — subtle outer glow **only** on accent focus/active states (buttons, inputs). No glow on every border.
- **No gradients behind text** unless contrast is verified (WCAG-minded spot check).
- **Motion restrained** — one animation vocabulary (prefer existing Framer Motion patterns, shortened).

## Collaboration model (V2.1)

### New table: `workspace_profiles`

| Column | Type | Notes |
|--------|------|--------|
| `id` | uuid PK | `gen_random_uuid()` |
| `workspace_id` | uuid FK → `workspaces` | cascade delete |
| `user_id` | uuid FK → `auth.users` | cascade delete |
| `display_name` | text NOT NULL | default `''` |
| `created_at` | timestamptz | default `now()` |
| **Unique** | | `(workspace_id, user_id)` |

**RLS (intent):**

- **SELECT:** any member of `workspace_id`.
- **INSERT:** via `ensure_profile_for_user(workspace_id)` security definer RPC **or** narrow policy (own row when member).
- **UPDATE:** `user_id = auth.uid()` for own `display_name`; **admin** may update any row in workspace.

### Tasks: stable assignee

- Add `assignee_user_id uuid NULL` → `auth.users(id)` on delete set null.
- Stop writing `assignee_member_id` from UI (column may remain for migration safety; deprecate).
- Assignee picker options = `workspace_profiles` for active workspace (`display_name` + `user_id`).

### RPC: `ensure_profile_for_user(p_workspace_id uuid)`

- Security definer; verify `auth.uid()` is member via `workspace_members`.
- Upsert profile row if missing (display_name default: email local-part or empty).
- Called on planner load after `fetchActiveWorkspaceContext`.
- Backfill existing members in migration (one-time definer function).

### Frontend

| File | Role |
|------|------|
| `src/lib/teamDb.ts` | `fetchProfiles`, `updateMyDisplayName`, `adminUpdateDisplayName` |
| `src/pages/PlannerPage.tsx` | Remove `teamMembers` / `localStorage` |
| `src/components/TeamPanel.tsx` | Workspace members + display name edit |
| `src/lib/plannerDb.ts` | `assignee_user_id` read/write |

**Remove:** all `localStorage` keys `teamMembers` and assignee flows using `TeamMember.id: number` from `Date.now()`.
